## Steps to use a custom configuration in Yuto Agentic

Once you've found the configuration you want to use on Yuto Agentic Mission Control:

1. Click “Add Config” on its page
2. Add any required inputs (e.g. secrets like API keys)

After saving, open the Yuto Agentic CLI or IDE extension and

- Select the configuration from the **config dropdown** in the Yuto Agentic extension
- Type `/config` and select the configuration in the Yuto Agentic CLI
- Begin using it for chat mode, agent mode, or other configured capabilities.
