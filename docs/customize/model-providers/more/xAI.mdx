## Configuration

<Tabs>
  <Tab title="YAML">
  ```yaml title="config.yaml"
  models:
    - name: <MODEL_NAME>
      provider: xai
      model: <MODEL_ID>
      apiKey: <YOUR_XAI_API_KEY>
  ```
  </Tab>
  <Tab title="JSON (Deprecated)">
  ```json title="config.json"
  {
    "models": [
      {
        "title": "<MODEL_NAME>",
        "provider": "anthropic",
        "model": "<MODEL_ID>",
        "apiKey": "<YOUR_XAI_API_KEY>"
      }
    ]
  }
  ```
  </Tab>
</Tabs>
