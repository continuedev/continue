## Configuration

<Tabs>
  <Tab title="YAML">
  ```yaml title="config.yaml"
  models:
    - name: <MODEL_NAME>
      provider: lmstudio
      model: <MODEL_ID>
      apiBase: http://<MY_ENDPOINT>/v1 # if running a remote instance of LM Studio
  ```
  </Tab>
  <Tab title="JSON (Deprecated)">
  ```json title="config.json"
  {
    "models": [
      {
        "title": "<MODEL_NAME>",
        "provider": "lmstudio",
        "model": "<MODEL_ID>",
        "apiBase": "http://<MY_ENDPOINT>/v1" // if running a remote instance of LM Studio
      }
    ]
  }
  ```
  </Tab>
</Tabs>

<Note>
  The default `apiBase` is `http://localhost:1234/v1`
</Note>
