</Info>

## Configuration

<Tabs>
  <Tab title="YAML">
  ```yaml title="config.yaml"
  models:
    - name: <MODEL_NAME>
      provider: inception
      model: <MODEL_ID>
      apiKey: <INCEPTION_API_KEY>
  ```
  </Tab>
  <Tab title="JSON (Deprecated)">
  ```json title="config.json"
  {
    "models": [
      {
        "title": "<MODEL_NAME>",
        "provider": "inception",
        "model": "<MODEL_ID>",
        "apiKey": "<INCEPTION_API_KEY>"
      }
    ]
  }
  ```
  </Tab>
</Tabs>
