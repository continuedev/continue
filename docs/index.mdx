## Continue CLI

<Info>Terminal-native AI coding assistance with TUI and headless modes.</Info>

<CodeGroup>

    ```bash npm
    npm install -g @continuedev/cli
    ```

    ```bash yarn
    yarn global add @continuedev/cli
    ```

    ```bash pnpm
    pnpm add -g @continuedev/cli
    ```

</CodeGroup>
