## Mission Control
<Note>
  Central dashboard for managing Agents, Tasks, Automations, and Integrations.
</Note>

  <Card title="Create a Task" icon="plus" href="/mission-control/tasks">

    <br />
    <sub>• Run manually or from your Inbox</sub><br />
    <sub>• Reuse and share with teammates</sub>

  <Card title="Set up an Automation" icon="clock" href="/mission-control/automations">

## Continue CLI

<Note>Terminal-native AI coding assistance with TUI and headless modes.</Note>

<CodeGroup>

    ```bash npm
    npm install -g @continuedev/cli
    ```

    ```bash yarn
    yarn global add @continuedev/cli
    ```

    ```bash pnpm
    pnpm add -g @continuedev/cli
    ```

</CodeGroup>
