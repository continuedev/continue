# Configuration Options

This page describes the configuration options available for Continue. You can set these options in two ways:

1. In a `config.json` file
2. In a `.continuerc.json` file at the root of your workspace

The `.continuerc.json` file can contain a partial configuration that will be applied on top of any settings from `config.json`.
