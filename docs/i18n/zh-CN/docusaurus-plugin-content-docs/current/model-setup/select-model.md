# 选择一个大语言模型

Continue 可以简单地切换不同的 LLM 。你可以点击模型下拉框旁边的 "+" 按钮在 GUI 中配置，或者手动添加到你的 `config.json` 中。一旦你完成这个，你可以在模型选择下拉框中切换它们。

**除了选择一个 LLM ，你还需要解决 [使用哪个模型提供者](./select-provider.md)。**
