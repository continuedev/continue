为了最好的聊天体验，我们推荐来自 Anthropic 的 Claude 3.7 Sonnet 。

从 hub [这里](https://hub.continue.dev/anthropic/claude-3-7-sonnet) 添加 Claude 3.7 Sonnet 到你的助手。你可以从 [Anthropic 控制台](https://console.anthropic.com/settings/keys) 获取一个 API key 。

其他推荐的聊天模型包括：

- [来自 OpenAI 的 GPT-4o](https://hub.continue.dev/openai/gpt-4o)
- [来自 xAI 的 Grok-2](https://hub.continue.dev/xai/grok-2)
- [来自 Google 的 Gemini 2.0 Flash](https://hub.continue.dev/google/gemini-2.0-flash)
- [Ollama 上的来自 Meta 的 Llama 3.1 8b](https://hub.continue.dev/ollama/llama3.1-8b) 获取完整的本地体验
