为了最好的自动补全体验，我们推荐来自 Mistral 的 Codestral 。

从 hub [在这里](https://hub.continue.dev/mistral/codestral) 添加 Codestral 到你的助手。访问 [Mistral Platform](https://console.mistral.ai/codestral) 获取一个 Codestral API key 。

为了完全本地自动补全体验，我们推荐 [Ollama 上的 Qwen 2.5 Coder 1.5B](https://hub.continue.dev/ollama/qwen2.5-coder-1.5b) 。

为了获取更多关于设置和推荐模型的详情，访问 [深入理解自动补全](../customize/deep-dives/autocomplete.mdx) 。
