当编辑代码时，聊天和编辑模型输出通常不能清楚地排列到现有代码中。一个 `应用` 角色的模型用来生成更简洁的 diff 来应用变更到文件中。

一般说来，在聊天和编辑中精通的模型也可以用来应用代码。

在 [hub](https://hub.continue.dev/explore/models?roles=apply) 中查看应用模型。
