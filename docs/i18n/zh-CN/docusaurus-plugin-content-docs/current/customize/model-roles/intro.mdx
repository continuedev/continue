在 Continue 中的模型可以配置用来扩展中不同的角色。

- [`chat`](./chat.mdx): 用来在扩展侧边栏中聊天交谈
- [`autocomplete`](./autocomplete.md): 用来在编辑器中自动补全代码建议
- [`edit`](./edit.mdx): 用来基于编辑提示词生成代码
- [`apply`](./apply.mdx): 用来决定如何应用编辑到文件中
- [`embed`](./embeddings.mdx): 用来生成向量搜索的嵌入 (@Codebase 和 @Docs 上下文提供者)
- [`rerank`](./reranking.mdx): 用来对向量搜索的结果重排序

这些角色可以使用 `roles` 指定为`config.yaml` 中的模型块。查看 [YAML 说明](../../yaml-reference.md#models) 获取更多详情。

## 选择模型角色

你可以控制在你的助手中给定角色的哪个模型当前用来那个角色。访问 [设置页面](../settings.md) 并去到 **Active Models** 章节。

![Settings Active Models Section](/img/settings-model-roles.png)

:::info
注意， `roles` 不存在在 `config.json` 中 - 它们由最高级的键引用，比如 `embeddingsProvider`
:::
