## 聊天模型

我们推荐配置 **Nemotron-4-340B-Instruct** 作为你的聊天模型。

<Tabs groupId="config-example">
  <TabItem value="yaml" label="YAML">
  ```yaml title="config.yaml"
  models:
    - name: Nemotron-4-340B-Instruct
      provider: nvidia
      model: nvidia-nemotron-4-340b-instruct
      apiKey: <YOUR_NVIDIA_API_KEY>
  ```
  </TabItem>
  <TabItem value="json" label="JSON">
  ```json title="config.json"
  {
    "models": [
      {
        "title": "Nemotron-4-340B-Instruct",
        "provider": "nvidia",
        "model": "nvidia-nemotron-4-340b-instruct",
        "apiKey": "<YOUR_NVIDIA_API_KEY>"
      }
    ]
  }
  ```
  </TabItem>
</Tabs>

## 自动补全模型

[点击这里](../../model-roles/autocomplete.md) 查看自动补全模型提供者列表。

## 嵌入模型

我们推荐配置 **NVIDIA Retrieval QA Mistral 7B** 作为你的嵌入模型。

<Tabs groupId="config-example">
  <TabItem value="yaml" label="YAML">
  ```yaml title="config.yaml"
  models:
    - name: Nvidia Embedder
      provider: nvidia
      model: nvidia/nv-embedqa-mistral-7b-v2
      apiKey: <YOUR_NVIDIA_API_KEY>
      roles:
        - embed
  ```
  </TabItem>
  <TabItem value="json" label="JSON">
  ```json title="config.json"
  {
    "embeddingsProvider": {
      "provider": "nvidia",
      "model": "nvidia/nv-embedqa-mistral-7b-v2",
      "apiKey": "<YOUR_NVIDIA_API_KEY>"
    }
  }
  ```
  </TabItem>
</Tabs>
