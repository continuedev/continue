# Function Network

> 私有的，可负担的，用户自己的 AI

:::info

为了获取一个 API key ，登录到 Function Network 开发者平台。如果你没有账号，可以在 [这里](https://www.function.network/join-waitlist) 创建一个。

:::

## 聊天模型

Function Network 支持多个聊天模型。我们推荐使用 LLama 3.1 70b 或 Qwen2.5-Coder-32B-Instruct 。

```json title="config.json"
{
  "models": [
    {
      "title": "Llama 3.1 70b",
      "provider": "function-network",
      "model": "meta/llama-3.1-70b-instruct",
      "apiKey": "YOUR_API_KEY"
    }
  ]
}
```

[点击这里](https://docs.function.network/models-supported/chat-and-code-completion) 查看聊天模型提供者列表。

## 自动补全模型

Function Network 支持多个自动补全模型。我们推荐使用 Llama 3.1 8b 或 Qwen2.5-Coder-1.5B 。

```json title="config.json"
{
  "tabAutocompleteModel": {
    "title": "Deepseek Coder 6.7b",
    "provider": "function-network",
    "model": "thebloke/deepseek-coder-6.7b-base-awq",
    "apiKey": "YOUR_API_KEY"
  }
}
```

## 嵌入模型

Function Network 支持多个嵌入模型。我们推荐使用 baai/bge-base-en-v1.5 。

```json title="config.json"
{
  "embeddingsProvider": {
    "provider": "function-network",
    "model": "baai/bge-base-en-v1.5",
    "apiKey": "YOUR_API_KEY"
  }
}
```

[点击这里](https://docs.function.network/models-supported/embeddings) 查看嵌入模型提供者列表。

## 重排序模型

Function Network 当前没有提供任何重排序模型。
