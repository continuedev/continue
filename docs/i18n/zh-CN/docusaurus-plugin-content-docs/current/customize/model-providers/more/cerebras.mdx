# Cerebras Inference

import TabItem from "@theme/TabItem";
import Tabs from "@theme/Tabs";

Cerebras Inference 使用专用的芯片提供快速推理。

1. 在门户 [这里](https://cloud.cerebras.ai/) 创建一个账户。
2. 创建和复制在 Continue 中使用的 API key 。
3. 更新你的 Continue 配置文件：

<Tabs groupId="config-example">
  <TabItem value="yaml" label="YAML">
  ```yaml title="config.yaml"
  models:
    - name: Cerebras Llama 3.1 70B
      provider: cerebras
      model: llama3.1-70b
      apiKey:  <YOUR_CEREBRAS_API_KEY>
  ```
  </TabItem>
  <TabItem value="json" label="JSON">
  ```json title="config.json"
  {
    "models": [
      {
        "title": "Cerebras Llama 3.1 70B",
        "provider": "cerebras",
        "model": "llama3.1-70b",
        "apiKey": "<YOUR_CEREBRAS_API_KEY>"
      }
    ]
  }
  ```
  </TabItem>
</Tabs>
