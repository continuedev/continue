## Continue CLI

<Note>Terminal-native AI coding assistance with TUI and headless modes.</Note>

<CodeGroup>

    ```bash npm
    npm i -g @continuedev/cli
    ```

    ```bash yarn
    yarn global add @continuedev/cli
    ```

    ```bash pnpm
    pnpm add -g @continuedev/cli
    ```

</CodeGroup>
