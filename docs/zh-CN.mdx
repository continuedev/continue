# 中文文档

## 欢迎

这是 Continue 项目的中文文档入口。

Continue 是一个开康的 AI 代码助手，支持 VS Code 和 JetBrains 扩展。

## 主要功能

- **聊天** - 与 AI 讨论代码
- **自动补全** - 智能代码建议  
- **编辑** - AI 辅助代码编辑
- **Agent** - AI 自动化开发任务

## 快速开始

1. 安装 VS Code 或 JetBrains 扩展
2. 配置模型
3. 开始使用
